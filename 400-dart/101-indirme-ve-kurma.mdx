## Chocolatey kullanarak kurulum
<br/>
* Dart SDK'yı yüklemek için [Chocolatey](https://chocolatey.org/) kullanın. Chocolatey [yükseltilmiş izinler](https://www.thewindowsclub.com/elevated-privileges-windows) gerektirir.

1. Chocolatey'i kurun.
2. PowerShell'i yükseltilmiş izinlerle başlatın.

```
PS C:\> choco install dart-sdk
```

<br />
## Varsayılan kurulum yolunu değiştir
<br />
* Varsayılan olarak, Chocolatey SDK'yı `C:\tools\dart-sdk` konumuna yükler. Bu konumu değiştirmek için
[ChocolateyToolsLocation](https://stackoverflow.com/questions/19752533/how-do-i-set-chocolatey-to-install-applications-onto-another-drive/68314437#68314437)
ortam değişkenini istediğiniz kurulum dizinine ayarlayın.
<br />
## PATH'inizin Dart'ı içerdiğini doğrulayın
<br />

- Dart'ı çalıştırabildiğinizi doğrulayın.

```PowerShell
PS dart --version
Dart SDK version: 3.2.4 (stable) (Thu Dec 21 19:13:53 2023 +0000) on "win_x64"
```

<br />

- Geliştirme makineniz bir Dart sürümü döndürmüyorsa, SDK konumunu PATH'inize ekleyin:

1. Windows arama kutusuna `env` yazın.
2. `Sistem ortam değişkenlerini düzenle`'ye tıklayın.
3. `Ortam Değişkenleri`'ne tıklayın....
4. Kullanıcı değişkeni bölümünde `Yol`'u seçin ve `Düzenle`'ye tıklayın....
5. `Yeni`'ye tıklayın ve dart-sdk dizinine giden yolu girin.
6. Az önce açtığınız her pencerede, kapatmak ve yol değişikliğini uygulamak için `Uygula` veya `Tamam`'a tıklayın.

## Chocolatey kullanarak yükseltme

<br />

- Dart SDK'yı yükseltmek için aşağıdaki komutu kullanın.

```PowerShell
PS choco upgrade dart-sdk
```

<br />

  ## Linux paket yöneticisini kullanarak kurulum yapın
  <br />
  * Dart SDK'yı Ubuntu veya Debian'a kurmak için iki seçeneğiniz var: 1. `apt-get` komutunu kullanın
  . 2. `.deb` paketi indirin ve `dpkg` komutu çalıştırın.
  <br />
  ## `apt-get` Paket yöneticisini kullanarak kurulum yapın
  <br />
  * Dart'ı `apt-get` ile kurmak için aşağıdaki adımları uygulayın.
  <br />
  1. Paket dizin dosyalarını güncelleyin ve güvenli HTTP paketini yükleyin. ```PowerShell sudo
  apt-get update && sudo apt-get install apt-transport-https ```
  <br />
  2. Google Linux GPG genel anahtarını indirin ve ekleyin. ``` wget -qO-
  https://dl-ssl.google.com/linux/linux_signing_key.pub \ | sudo gpg --dearmor -o
  /usr/share/keyrings/dart.gpg ```
  <br />
  3. Dart paket deposunu Linux sisteminize ekleyin. ```PowerShell echo 'deb
  [signed-by=/usr/share/keyrings/dart.gpg arch=amd64]
  https://storage.googleapis.com/download.dartlang.org/linux/debian stable main' \ | sudo tee
  /etc/apt/sources.list.d/dart_stable.list ```
  <br />
  4. Aşağıdaki sudo apt-get komutlarını kullanın. ```PowerShell sudo apt-get update && sudo apt-get
  install dart ```
  <br />
  ## Debian paketi olarak kurun
  <br />
  * Dart SDK'yı Debian paketi `(*.deb)` olarak yüklemek için aşağıdaki adımları uygulayın. 1. Dart
  SDK
  [Debian](https://storage.googleapis.com/dart-archive/channels/stable/release/latest/linux_packages/dart_3.6.0-1_amd64.deb)
  paketini indirin. 2. `*.deb` paketini yüklemek için `sudo dpkg` komutunu kullanın. ```PowerShell
  sudo dpkg -i dart_3.4.0-1_amd64.deb ```
  <br />
  dart_3.4.0-1_amd64.deb dosyasını geçerli dosya adıyla değiştirin.
  <br />
  ## Dart SDK'sını yükseltin
  <br />
  * SDK'yi kurmak için kullandığınız komutun aynısını kullanın. ### `apt-get` kullanarak yükseltme *
  Dart SDK'yı apt-get ile yüklediyseniz, aşağıdaki sudo apt-get komutlarını kullanın. ```PowerShell
  sudo apt-get update && sudo apt-get install dart ```
  <br />
  ### dpkg kullanarak yükseltme
  <br />
  * Dart SDK'yı `dpkg` ile yüklediyseniz `sudo dpkg` komutunu kullanın. ```PowerShell sudo dpkg -i
  dart_3.2.6-1_amd64.deb ``` dart_3.4.0-1_amd64.deb dosyasını yeni yükseltmenin dosya adıyla
  değiştirin.
  <br />
  ## Dart SDK'yi kaldırın
  <br />
  ### `apt-get` kullanarak kaldır
  <br />
  * Dart SDK'yı `apt-get` ile yüklediyseniz, `sudo apt-get remove` komutunu kullanın.
  <br />
  1. sudo apt-get remove komutunu kullanın.
  <br />
  ```PowerShell sudo apt-get remove -y dart ```
  <br />
  2. Dart yapılandırma dosyalarını ana dizininizden kaldırın.
  <br />
  ```PowerShell rm -rf ~/.dart* ```
  <br />
  ### `dpkg` kullanarak kaldır
  <br />
  * Dart SDK'yı `dpkg` ile yüklediyseniz `sudo dpkg --purge` komutunu kullanın.
  <br />
  1. `sudo dpkg --purge` komutunu kullanın.
  <br />
  ```PowerShell sudo dpkg --purge dart ``` Bu işlem aynı zamanda yapılandırma dosyalarını da
  kaldırır.
  <br />
  2. SDK'nın kaldırıldığını doğrulayın
  <br />
  ```PowerShell dpkg -l | grep dart ```

<TabItem value="macOS"> 
## Homebrew kullanarak kurulum
<br/>
### Dart SDK'yı kurmak için [Homebrew](https://brew.sh/)'u kullanın.
<br/>
1. Gerekirse Homebrew'u yükleyin.
2. [official tap](https://github.com/dart-lang/homebrew-dart) i ekleyin.
<br/>
```
brew tap dart-lang/dart
```
<br/>
3. Dart SDK’yı yükleyin.
<br/>
```
brew install dart
```
<br/>
## PATH'in Homebrew'u içerdiğini doğrulayın
<br/>
* PATH'inizin **Homebrew** bin dizinini içerdiğini doğrulayın. `Dart run` ve `dart format` gibi Dart SDK komutlarını kullanarak doğru yolu ayarlamak basitleşir.
<br/>
* PATH'inizi yapılandırma konusunda yardım almak için [Homebrew SSS](https://docs.brew.sh/FAQ#my-mac-apps-dont-find-homebrew-utilities)'ye bakın.
<br/>
## Homebrew kullanarak yükseltme
<br/>
* Dart'ın yeni bir sürümü çıktığında güncelleme yapmak için:
<br/>
```
brew upgrade dart
```
<br/>
## Dart sürümlerini değiştir
<br/>
* Yerel olarak yüklenen Dart sürümleri arasında geçiş yapmak için:
<br/>
1. Geçmek istediğiniz sürümü yükleyin.
  - Örneğin Dart 3.1'i yüklemek için:
<br/>
```
brew install dart@3.1
```
<br/>
2. Sürümler arasında geçiş yapmak için geçerli sürümü bağlantısını kaldırın ve istediğiniz sürümü bağlayın.
<br/>
```
brew unlink dart@<old> \
  && brew unlink dart@<new> \
  && brew link dart@<new>
```
<br/>
## Kurulu Dart sürümlerini listele
<br/>
* Hangi Dart sürümlerini yüklediğinizi görmek için:
<br/>
```
brew info dart
```
<br/>
## Homebrew kullanarak kaldırma
<br/>
* Dart SDK'yı kaldırmak için [Homebrew](https://brew.sh/)'u kullanın.
<br/>
1. Dart SDK'yı kaldırın.
<br/>
```
brew uninstall dart
```
<br/>
2. Dart yapılandırma dosyalarını ana dizininizden kaldırın.
<br/>
```
rm -rf  ~/.dart*
```
